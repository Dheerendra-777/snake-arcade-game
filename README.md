# 🐍 Snake Arcade Game

A modern and advanced Snake Game built using **HTML, CSS, and JavaScript**.

This project is a fully interactive browser game with multiple themes, power-ups, difficulty levels, and leaderboard functionality.

---

## 🎮 Features

- Multiple Themes (Dark, Light, Retro Arcade)
- Difficulty Levels (Beginner, Intermediate, Professional)
- Power-Ups System
- Leaderboard System
- Individual & Team Play Mode
- Sound Effects
- Mobile Controls Support
- Dynamic Obstacles
- Level System
- Combo Score System
- Fullscreen Gameplay

---

## 🕹️ How to Play

1. Open the game in your browser
2. Select your preferred settings
3. Click **Play Fullscreen**
4. Control the snake using:

Arrow Keys OR WASD Keys
Spacebar → Pause Game

Goal:  
Eat food to grow the snake and increase your score. Avoid hitting walls, obstacles, or your own tail.

---

## ⚡ Power-Ups

| Power-Up | Description |
|--------|-------------|
| Golden Apple | Gives 4x points |
| Blue Crystal | Slows down the game |
| Purple Orb | Attracts food toward the snake |
| Poison | Reduces score |
